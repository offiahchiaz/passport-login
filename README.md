# Passport Login
This is a user login and registration app using Node.js, Express, Passport, Mongoose, EJS and some other packages.
### Usage
$ npm install
```
$ npm install
```

```
$ npm start
# Or run with Nodemon
$ npm run dev

# Visit http://localhost:5000
```
### MongoDB
Open "config/keys.js" and add your MongoDB URI, local or mlab.
